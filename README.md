# a2-code-prototype
Instructions for the use of the breathing training program

Project Introduction
This project includes two patterns: Breathing Training and Free Running which run on the Micro:Bit.

Operation Guide
Basic Operation

Press A: mode selection/Game Jump
Press B: Training Start/Terminate training 
Touch Logo: pattern switching


Breathing Training Pattern
After the device is start, it enters the breathing training mode firstly, and the screen shows "Training"

When you press A to switch between breathing training modes (NORMAL,RELAX,FOCUS), press B to start the training.

The prompt sound will be played during the training.

Press the B button during the training to finish at any time.

After the training, the screen will display the training score (i.e., the number of breaths)

Game Pattern
Touch the logo to enter the game mode

The screen shows the game name "Free Running"

Please wait for the full display of the letters before pressing the button

Press the B button to start the game

Game operation

Press the A button or make an exhale to control the player to jump

Avoid obstacles by jumping. If the player touched obstacles, "Game Over" will be triggered.

It displays the final score on the screen after the game was over.


Pattern Switching

After completing the game, the screen will become blank

Touch the logo again to return to the breathing training pattern


Precautions
Please make sure that the Micro: Bit is energised normally

In the game pattern, please wait for the game name to be fully displayed before operating

You can press the B button to exit at any time during the training
